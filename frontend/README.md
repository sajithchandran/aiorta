# Frontend App

This folder contains the frontend application scaffold for the research workspace.

Current layout:
- `src/app`: future Next.js App Router routes
- `src/features`: feature-oriented UI slices
- `src/components`: reusable UI areas
- `src/lib`: typed client helpers
- `src/providers`: app-level providers
